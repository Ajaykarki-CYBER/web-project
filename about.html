About This Project

What We Did

We created a simple and clean To-Do List Web App using only HTML and CSS. The purpose was to design a practical and visually appealing tool that helps users organize their daily tasks in a neat format.

What We Added
Task creation with input fields
Category selection (Work, Study, Personal)
Priority levels (Low, Medium, High)
Task cards with deadlines and status
Report page summarizing completed and pending tasks
